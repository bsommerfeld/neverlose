## Why Neverlose

- Lightning‑fast plan editor
- Reusable drills and training units
- One‑click, professional PDF export
- Cross‑platform: Windows, macOS, Linux

## Get Started

1. Download the latest version from the [Releases](https://github.com/bsommerfeld/neverlose/releases) page.
2. Launch the app.
3. Create your first plan, save favorite drills, and export to PDF.
